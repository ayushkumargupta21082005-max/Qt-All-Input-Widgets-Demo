# Qt All Input Widgets Demo

A beginner-friendly desktop application built using **Qt Widgets** and **C++**.

This project demonstrates the implementation and usage of the most commonly used Qt input widgets in a single application.

---

## Features

✅ Combo Box (Fruit Selection)

✅ Font Combo Box

✅ Line Edit (Name)

✅ Password Field

✅ Text Edit (Query)

✅ Spin Box (Age)

✅ Double Spin Box (Salary)

✅ Time Edit

✅ Date Edit

✅ Date Time Edit

✅ Dial

✅ Scroll Bar

✅ Vertical Slider

✅ Key Sequence Edit

✅ Push Button (Fetch)

✅ Dynamic Output Panel

## How to Run

1. Clone the repository

```bash
git clone https://github.com/yourusername/Qt-All-Input-Widgets-Demo.git
```

2. Open **allinputwidets.pro** using Qt Creator.

3. Build the project.

4. Run the application.
